## How to Use This Document

1. Prefer a short **reproduction recipe** and the Compose profile (`--profile`) or environment used.
2. Tag the **area** (`gateway`, `ingestion`, `detection`, `ai`, `dashboard`, `infra`).
3. Link a **metric name** or structured **log fingerprint** where possible.

When an issue is fixed, move the row to a **Resolved** subsection with PR link and date.
